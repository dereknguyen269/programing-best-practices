# AI Agent Instructions

## Repository Overview

This is a **Programming Best Practices Knowledge Base** - a curated collection of coding standards, style guides, and best practices across 30+ programming languages and frameworks.

## Primary Document

**README.md** is the main knowledge base containing all curated resources organized by:
- Backend Development (C, C++, Rust, Java, Kotlin, Python, Ruby, Go, etc.)
- Frontend Development (HTML, CSS, JavaScript, TypeScript, React, Vue, Angular)
- Database & Data (SQL, PostgreSQL, MySQL, NoSQL)
- Mobile Development (Flutter, Dart, React Native)
- DevOps & Infrastructure (AWS, Microservices, Security)
- AI & Data Science (MLOps, AI Tools)
- Development Tools (Code Review, Bash, System Design)

## How to Use This Knowledge Base

### For Code Quality Questions
When asked about coding standards or best practices:
1. Reference the specific language section in README.md
2. Cite the original source links provided
3. Prioritize resources from well-known organizations (Google, Airbnb, Uber)

### For Architecture Questions
- Reference the System Design section
- Use The Twelve-Factor App for cloud-native applications
- Check Microservices best practices for distributed systems

### For Security Questions
- Reference API Security and DevSecOps sections
- Cite OWASP Top 10 for web security
- Check language-specific security guides when available

### For Code Reviews
- Use featured resources as quality benchmarks
- Apply language-specific style guides
- Reference Clean Code principles
